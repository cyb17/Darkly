# Darkly

Darkly is a web security project designed to introduce common web vulnerabilities and attack techniques.  
The goal is to analyze insecure web application behaviors, exploit vulnerabilities, and understand how to properly mitigate them.

## Common web vulnerabilities explored in this project

- SQL Injection (SQLi)
- Cross‑Site Scripting (XSS)
- Path Traversal
- Local File Inclusion (LFI)
- Remote File Inclusion (RFI)
- Brute Force Attacks
- Broken Authentication
- Broken Access Control
- Security Misconfiguration
- Information Disclosure
- Parameter Tampering
- Insecure Direct Object References (IDOR)
- Client‑Side Trust Issues
- Business Logic Flaws
- Cross‑Site Request Forgery (CSRF)
